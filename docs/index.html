我们的故事

初识于高中，但真正的走进对方的生活源于在北京的一次偶然相遇。因为一起帮朋友照顾一只叫“两块”的土狗，让我们走到了一起。在北京住着不大的两居室，努力的奋斗着。生活于我们而言从来不会平淡，酸甜苦辣咸，应有尽有，但最终总会沉淀出属于我们自己的故事和人生旅途。
